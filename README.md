## 🛠️ Local Development & Deployment

### Prerequisite
Ensure you have [Node.js](https://nodejs.org) installed on your system.

### 1. Clone the Codebase
```bash
git clone https://github.com/Adzumr/adzumr-portfolio.git
cd adzumr-portfolio
```

### 2. Install Development Dependencies
```bash
npm install
```

### 3. Run Local Dev Server (Vite)
To start a high-performance local server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173`.

### 4. Build for Production
To bundle and optimize all files for static deployment:
```bash
npm run build
```

### 5. Deploy to Firebase Hosting
Deploying changes is fully automated via the Firebase CLI:
```bash
# Log in to your Firebase account
npx firebase login

# Deploy static assets directly to the production site
npx firebase deploy --project adzumr-portfolio
```
Once complete, the site will be live at [https://adzumr-portfolio.web.app](https://adzumr-portfolio.web.app).
